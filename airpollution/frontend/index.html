Airquality-Index Predicter

Instructions

Enter latitude and longitude or drag the map to get a prediction about the airquality of the given location.





Additional Infos

The Data used was collected beforehand and doesn't represent the present datetime.
All Data is collected from: waqi.info
Predictions with an index of 0 are to be interpreted carefully. A value of 0 can be explained by no nearby airscanners.
